# Проект "Сервис - UrlShortCut"

## Описание проекта.

RСервис работает через REST API.

Требуемый функционал.

1. Регистрация сайта.
2. Авторизация.
3. Регистрация URL.
4. Переадресация. Выполняется без авторизации.
5. Статистика.

## Стек технологий.

- **Java 17**
- **Spring Boot 2.7.10**
- **PostgreSQL 14**
- **lombok**


## Требования к окружению.

- **Java 17**
- **Maven 3.8**
- **PostgresSQL 14**
